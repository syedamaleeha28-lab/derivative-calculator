## About

Calculadora Derivadas is an educational mathematics tool designed to make differential calculus easier to understand. Users can enter functions and receive accurate derivative results along with detailed step-by-step explanations.

The platform supports a wide range of mathematical expressions, helping users learn and verify derivative calculations efficiently.
